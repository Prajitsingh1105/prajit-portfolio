# Portfolio

Welcome to my personal portfolio! This repository houses the source code for my professional site, designed to showcase my journey as a Computer Science student at IET Lucknow and my work as a full-stack developer.

## 🚀 About
I specialize in building high-performance web applications using the **MERN stack** (MongoDB, Express, React, Node.js). My design philosophy centers on clean, functional, and "premium" UI/UX—often inspired by modern SaaS aesthetics like glassmorphism and minimalist grid systems.

## 🛠 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Design:** Figma, Framer Motion (for animations)

## 📂 Featured Projects
- **[Sprintern]:** A centralized internship platform compliant with NEP 2020 guidelines.
- **[Flasheats]:** A smart campus canteen food ordering system featuring real-time dashboards.
- **[IET Placement Portal]:** UI/UX overhaul of the Training & Placement portal, focusing on recruiter interaction.

## 🎨 Design Philosophy
My projects prioritize user experience through:
* **Minimalism:** Keeping interfaces clean and distraction-free.
* **Modern Effects:** Utilizing frosted glass (glassmorphism) and 3D isometric elements for a polished, high-end feel.
* **Responsiveness:** Ensuring a seamless experience across all device sizes.
